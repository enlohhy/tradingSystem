==========================================
  加密货币实时监测系统 - 项目说明
==========================================

【项目概述】
这是一个用Python开发的加密货币实时价格监测应用，可以监测BTC、ETH、SOL、DOGE等主流币种的价格，并以文字形式实时展示。

【主要功能】
✓ 实时获取加密货币价格（来自CoinGecko API）
✓ 显示24小时涨跌幅、交易量、市值
✓ 彩色终端界面，涨跌一目了然
✓ 自动刷新，可配置刷新间隔
✓ 支持添加更多币种
✓ 跨平台支持（Windows/Linux/macOS）

【文件结构】
detection/
├── cli_app.py              # 主应用程序
├── requirements.txt        # 依赖包列表
├── README.md              # 详细文档
├── 快速开始.md             # 快速入门指南
├── 启动应用.bat            # Windows启动脚本
├── config/                # 配置目录
│   └── config.yaml        # 配置文件
└── src/                   # 源代码
    ├── data_fetcher.py    # 数据获取模块
    └── config_loader.py   # 配置加载模块

【快速使用】
1. 安装依赖：pip install -r requirements.txt
2. 运行应用：
   - Windows: 双击 "启动应用.bat"
   - 其他: python cli_app.py
3. 按 Ctrl+C 退出

【依赖包】
- requests (HTTP请求)
- pyyaml (配置文件解析)
- colorama (彩色终端输出)

【配置说明】
编辑 config/config.yaml 可以：
- 修改监测币种
- 调整刷新间隔
- 更改显示货币（USD/CNY等）

【数据来源】
CoinGecko API (免费版)
- 实时价格数据
- 24小时交易数据
- 市值信息

【显示说明】
▲ 绿色 = 价格上涨
▼ 红色 = 价格下跌
= 黄色 = 价格持平

【注意事项】
1. 需要稳定的网络连接
2. API有速率限制（50次/分钟）
3. 数据仅供参考，不构成投资建议
4. Windows用户如遇乱码请使用提供的bat脚本启动

【技术特点】
