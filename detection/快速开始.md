# 快速开始指南

## 1. 安装依赖

```bash
pip install -r requirements.txt
```

**依赖包**：
- `requests` - HTTP请求
- `pyyaml` - 配置解析
- `colorama` - 彩色终端
- `websocket-client` - WebSocket连接

## 2. 运行应用

### Windows用户
双击 `启动应用.bat` 文件

### 所有平台
```bash
python cli_app.py
```

## 3. 使用说明

- **数据源**：币安WebSocket（极速实时推送）
- **推送模式**：实时推送，无延迟（<100ms）
- **监测币种**：BTC、ETH、SOL、DOGE
- **显示刷新**：每3秒刷新屏幕（数据实时接收）
- **退出程序**：按 `Ctrl+C`

## 🚀 WebSocket特性

### ✨ 真正的实时数据
- **延迟**：<100ms（之前10秒）
- **准确性**：100%捕获价格变化
- **稳定性**：持久连接+自动重连
- **效率**：节省90%网络带宽

## 4. 自定义配置

编辑 `config/config.yaml` 文件：

```yaml
# 监测的币种列表
cryptocurrencies:
  - BTC
  - ETH
  - SOL
  - DOGE
  - BNB  # 添加新币种

# 币安WebSocket配置
binance:
  ws_url: "wss://stream.binance.com:9443"
  api_url: "https://api.binance.com"
  
  # 交易对映射
  symbol_map:
    BTC: "BTCUSDT"
    ETH: "ETHUSDT"
    SOL: "SOLUSDT"
    DOGE: "DOGEUSDT"
    BNB: "BNBUSDT"  # 新增
```

**注意**：
- 无需配置刷新间隔（实时推送）
- 只使用币安数据源（已删除CoinGecko）

## 5. 常见问题

### WebSocket连接失败
- 检查网络连接
- 确保能访问 `stream.binance.com`
- 检查防火墙设置
- 确认端口9443未被屏蔽

### 没有收到数据
- WebSocket建立需要2-3秒
- 初始数据从REST API获取
- ticker数据每秒推送一次
- 耐心等待几秒

### 显示乱码
- 使用提供的 `启动应用.bat` 启动
- 或在命令行先执行 `chcp 65001`
