# 📱 移动端适配说明

## ✅ 已完成的优化

### 1. 侧边栏抽屉式菜单
- **问题**：在移动端（屏幕宽度 < 1024px）时，侧边栏完全隐藏，无法访问币种列表
- **解决方案**：
  - 添加了汉堡菜单按钮（☰）在导航栏左侧
  - 侧边栏以滑动抽屉的形式从左侧滑入
  - 添加了半透明遮罩层，点击可关闭侧边栏
  - 选择币种后自动关闭侧边栏

### 2. 响应式断点优化

#### 📐 屏幕宽度 ≤ 1024px（平板和手机）
- 侧边栏改为固定定位的抽屉式菜单
- 主容器改为单列布局
- 显示汉堡菜单按钮
- Logo 字体缩小至 18px

#### 📱 屏幕宽度 ≤ 768px（手机）
- 导航栏高度从 60px 调整为 56px
- 价格卡片改为单列显示
- 内容区域 padding 缩小为 12px
- 隐藏连接状态指示器（节省空间）
- 导航按钮文字保留（带 span 标签包裹）
- Logo 字体进一步缩小至 16px

#### 📱 屏幕宽度 ≤ 480px（小屏手机）
- 导航按钮只显示图标，隐藏文字
- 按钮尺寸调整为 36px × 36px
- 侧边栏宽度改为 100%（全屏）
- 模态框宽度增加至 95%
- 所有卡片、表单、文字大小优化
- 共振分析网格改为单列

### 3. 交互体验优化
- **防止背景滚动**：侧边栏打开时，背景内容不能滚动
- **平滑动画**：侧边栏滑入滑出使用 0.3s 过渡动画
- **触摸友好**：所有按钮和交互元素都有合适的触摸区域
- **自动关闭**：选择币种后侧边栏自动关闭，提升体验

### 4. 视觉优化
- 遮罩层半透明黑色背景（rgba(0, 0, 0, 0.5)）
- 侧边栏带有阴影效果，增加层次感
- 按钮按下时有缩放动画（scale(0.95)）
- 保持币安暗色主题风格

## 🎯 测试建议

### 在不同设备上测试
1. **桌面浏览器**（宽度 > 1024px）
   - 侧边栏应该正常显示在左侧
   - 汉堡菜单按钮应该隐藏

2. **平板设备**（768px - 1024px）
   - 点击汉堡菜单按钮，侧边栏从左侧滑入
   - 点击遮罩层或选择币种后，侧边栏自动关闭
   - 侧边栏宽度 280px

3. **手机设备**（< 768px）
   - 导航栏更紧凑
   - 所有功能按钮正常显示
   - 价格卡片单列显示

4. **小屏手机**（< 480px）
   - 导航按钮只显示图标
   - 侧边栏全屏显示
   - 所有文字和间距适配小屏幕

## 📝 主要代码改动

### CSS 改动
- 新增 `.mobile-menu-btn` 样式（汉堡菜单按钮）
- 新增 `.sidebar-overlay` 样式（遮罩层）
- 优化 3 个响应式断点的媒体查询
- 新增移动端特定样式（字体、间距、布局）

### JavaScript 改动
- 新增 `toggleSidebar()` 函数（切换侧边栏）
- 新增 `closeSidebar()` 函数（关闭侧边栏）
- 修改 `selectCoin()` 函数（添加自动关闭逻辑）
- 修改 `toggleMode()` 函数（更新按钮 HTML 结构）

### HTML 改动
- 导航栏添加汉堡菜单按钮
- 添加侧边栏遮罩层元素
- 按钮文字用 `<span>` 标签包裹（便于移动端隐藏）

## 🚀 使用方法

1. 在手机浏览器中打开 `index.html`
2. 点击左上角的 **☰** 按钮打开币种列表
3. 选择要查看的币种
4. 侧边栏会自动关闭，显示所选币种的详细信息
5. 需要切换币种时，再次点击 **☰** 按钮
