# 📱 移动端快速测试指南

## 🔧 方法一：使用浏览器开发者工具（推荐）

### Chrome / Edge
1. 打开 `index.html` 文件
2. 按 `F12` 打开开发者工具
3. 按 `Ctrl + Shift + M` （或点击工具栏的设备图标 📱）
4. 在顶部选择设备：
   - **iPhone SE** (375 × 667) - 小屏手机
   - **iPhone 12 Pro** (390 × 844) - 标准手机
   - **iPad Air** (820 × 1180) - 平板
   - **响应式** - 手动拖动调整大小

### Firefox
1. 打开 `index.html` 文件
2. 按 `F12` 打开开发者工具
3. 按 `Ctrl + Shift + M` 进入响应式设计模式
4. 选择设备预设或自定义尺寸

## 🎯 核心功能测试

### 1. 侧边栏菜单
```
✅ 测试步骤：
1. 点击 ☰ 按钮 → 侧边栏滑入
2. 点击遮罩层 → 侧边栏关闭

4. 点击任意币种 → 侧边栏自动关闭
5. 页面显示所选币种的信息
```

### 2. 币种切换
```
✅ 测试步骤：
1. 打开侧边栏
2. 点击 "BTCUSDT"
3. 侧边栏关闭，显示 BTC 价格和分析

5. 点击 "ETHUSDT"
6. 侧边栏关闭，显示 ETH 价格和分析

```

## 🐛 常见问题排查

### 问题1：看不到币种列表
- **原因**：侧边栏隐藏了
- **解决**：点击左上角的 ☰ 按钮

### 问题2：侧边栏打不开
- **原因**：JavaScript 未加载
- **解决**：刷新页面（F5），查看浏览器控制台是否有错误

### 问题3：按钮太小点不准
- **检查**：是否在正确的设备尺寸下测试
- **解决**：按钮已设置最小 36px × 36px 的触摸区域

### 问题4：布局错乱
- **原因**：缓存问题
- **解决**：按 Ctrl+F5 强制刷新页面

## 📱 真机测试（可选）

### 方法1：本地网络测试
1. 确保手机和电脑在同一 WiFi 网络
2. 在电脑上运行简单的 HTTP 服务器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (需要先安装 http-server)
   npx http-server -p 8000
   ```
3. 获取电脑的局域网 IP 地址（如：192.168.1.100）
4. 在手机浏览器访问：`http://192.168.1.100:8000/index.html`

### 方法2：文件传输
1. 将 `index.html` 文件传输到手机
2. 使用手机浏览器直接打开文件
