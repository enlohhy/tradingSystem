### 1. **⚠️ 智能分歧检测系统**

当"多周期共振分析"和"技术分析报告"出现不一致时，系统会：
- ✅ 自动检测并高亮显示分歧

| 状态 | 多周期共振 | 技术分析 | 显示 | 说明 |
|-----|-----------|---------|------|------|
| **强烈看涨** | 🟢 看涨 | 🟢 看涨 | 🟢 强烈一致 | 两个分析均看涨，高概率上涨机会 |
| **强烈看跌** | 🔴 看跌 | 🔴 看跌 | 🔴 强烈一致 | 两个分析均看跌，建议规避或做空 |

| **长跌短涨** | 🔴 看跌 | 🟢 看涨 | 🔷 分歧：长跌短涨 | 长期下跌，短期反弹，谨慎追高 |

### 2. **📊 优化多周期共振判断逻辑**

#### 旧版本（简单）
- 只看 MA20 和 MA60 两条均线
- 判断标准单一
- 容易误判

#### 新版本（智能）
使用多维度综合判断：

##### 均线排列（MA7、MA30、MA60）
```
看涨：当前价 > MA7 > MA30 > MA60
看跌：当前价 < MA7 < MA30 < MA60
```

##### RSI过滤（币圈标准）
```
超卖：RSI < 20
超买：RSI > 80
正常：20 ≤ RSI ≤ 80
```

**作用**：
- 过滤虚假信号：即使均线多头，但RSI超买则降低信号强度
- 确认真实机会：均线多头 + RSI未超买 + 量能配合 = 强烈买入信号

##### 成交量确认
```
量能强：近5期平均量 / 近20期平均量 > 1.2
量能弱：近5期平均量 / 近20期平均量 < 0.8
```

**作用**：
- 有效突破：价格上涨 + 成交量放大 = 真突破
- 虚假突破：价格上涨 + 成交量萎缩 = 假突破

#### 信号强度分级
```
最强信号：均线多头 + RSI正常 + 量能配合 → 强度 100%
强信号：  均线多头 + RSI正常          → 强度 80%
弱信号：  均线多头 + RSI超买          → 强度 60%
中性：    均线混乱                   → 强度 50%
```

### 3. **🔢 MA周期优化**

#### 修改前
```
MA5、MA10、MA20、MA30、MA60
```

#### 修改后（更适合币圈）
```
MA7、MA30、MA60、MA120、MA240
```

**优势**：
- MA7：一周行情，短期趋势
- MA30：一月行情，中期趋势
- MA60：两月行情，中长期趋势
- MA120：四月行情，长期趋势
- MA240：半年行情，超长期趋势

## 📖 使用指南

### 如何正确使用分歧提示

#### Step 1：执行技术分析
1. 选择要分析的币种
2. 点击"📊 技术分析"按钮
3. 系统自动执行多周期共振分析和技术分析

#### Step 2：查看分歧提示
在技术分析报告顶部查看：
- **信号一致性**：一致/分歧
- **信号强度**：0-100%
- **市场状态**：当前所处的市场阶段
- **操作建议**：具体的交易建议

#### Step 3：结合多周期共振详情
查看每个时间周期的具体数据：
- 趋势方向
- RSI值
- 成交量比率
- 触发信号

#### Step 4：做出交易决策

**决策矩阵**：

| 信号强度 | 一致性 | 建议操作 | 仓位 |
|---------|-------|---------|------|
| ≥80% | 一致 | 积极进场 | 20-30% |
| 60-79% | 一致 | 逐步建仓 | 10-20% |
| 50-79% | 分歧 | 轻仓试探 | 5-10% |
| <50% | 分歧 | 观望等待 | 0% |

### 案例1：强烈看涨信号

**场景**：
- 多周期共振：80% 看涨（4个周期看涨）
- 技术分析：85分 看涨

**系统提示**：
```
✅ 信号强烈一致 - 🟢 强烈一致

市场状态：多周期和技术指标均看涨，趋势向上确认

```

**操作**：
1. 确认当前价格位置
2. 设置止损（建议止损位）
3. 分批建仓 20-30%
4. 设置止盈目标

---

### 案例2：长涨短跌分歧

**场景**：
- 多周期共振：70% 看涨（3-4个周期看涨）
- 技术分析：65分 看跌（RSI超买、MACD死叉）

**系统提示**：
```
⚠️ 检测到信号分歧 - 🔶 分歧：长涨短跌
信号强度：67%
市场状态：长期趋势向上，但短期技术指标显示回调信号
操作建议：⚠️ 建议：暂缓买入，等待回调到支撑位；持仓者可以部分止盈
```

**操作**：
1. **如果空仓**：等待回调到支撑位（如MA30或斐波那契支撑位）
2. **如果持仓**：部分止盈（如30-50%），保护利润
3. **设置提醒**：价格回调到支撑位时提醒
4. **回调后**：当技术分析也转为看涨时，再次买入

---

### 案例3：长跌短涨分歧

**场景**：
- 多周期共振：60% 看跌（3个周期看跌）
- 技术分析：70分 看涨（RSI超卖反弹）

**系统提示**：
```
⚠️ 检测到信号分歧 - 🔷 分歧：长跌短涨
信号强度：58%
市场状态：长期趋势向下，短期技术指标显示反弹信号
操作建议：⚠️ 建议：反弹不是反转，谨慎追高，注意止盈
```

**操作**：
1. **小仓位参与**：最多5-10%仓位
2. **严格止损**：反弹失败立即止损
3. **快进快出**：反弹5-10%及时止盈
4. **避免贪婪**：反弹不是反转，不要期待太高
