### 2. 📊 新增VOL成交量指标
**重要指标**：成交量是价格变动的动力源泉！

#### VOL指标功能
- **当前量** - 最新的成交量
- **平均量** - 近期平均成交量
- **量比** - 当前量与平均量的比值
- **状态** - 量能状态判断

#### 量能状态
| 量比 | 状态 | 信号 | 含义 |
|------|------|------|------|
| >2.0x | 放量突破 | 买入 | 强烈买卖意愿 |
| 1.5-2.0x | 温和放量 | 买入 | 趋势确认 |
| 0.7-1.5x | 量能平稳 | 中性 | 正常波动 |
| 0.5-0.7x | 量能萎缩 | 中性 | 交易冷清 |
| <0.5x | 缩量明显 | 卖出 | 趋势疲软 |

## 🆕 VOL成交量指标详解

### 指标意义
成交量是**价格变动的驱动力**，VOL指标帮助：
- ✅ 确认趋势真实性
- ✅ 判断买卖力量
- ✅ 识别主力行为
- ✅ 预警趋势反转

### 使用方法

#### 1. 价涨量增 → 上涨趋势健康
```
价格：上涨
VOL：放量（>1.5x）
信号：买入 ✅
操作：持有或加仓
```

#### 2. 价涨量缩 → 上涨动能不足
```
价格：上涨
VOL：缩量（<0.7x）
信号：警惕 ⚠️
操作：考虑减仓
```

#### 3. 价跌量增 → 下跌趋势确立
```
价格：下跌
VOL：放量（>1.5x）
信号：卖出 ❌
操作：空仓观望
```

#### 4. 价跌量缩 → 下跌动能减弱
```
价格：下跌
VOL：缩量（<0.7x）
信号：关注 👀
操作：可能止跌
```

### VOL配合其他指标

#### VOL + MACD
```
MACD金叉 + VOL放量 → 强烈买入信号
MACD死叉 + VOL放量 → 强烈卖出信号
```

#### VOL + RSI
```
RSI超卖 + VOL放量 → 反弹机会
RSI超买 + VOL放量 → 顶部信号
```

#### VOL + KDJ
```
KDJ低位金叉 + VOL放量 → 上涨起点
KDJ高位死叉 + VOL放量 → 下跌起点
```

## ⚙️ 技术细节

### VOL计算公式
```javascript
当前量 = volumes[最新]
平均量 = sum(volumes[-5:]) / 5
量比 = 当前量 / 平均量

状态判断：
  量比 > 2.0  → 放量突破（买入）
  量比 > 1.5  → 温和放量（买入）
  量比 < 0.5  → 缩量明显（卖出）
  量比 < 0.7  → 量能萎缩（中性）
  其他        → 量能平稳（中性）
```

### 权重计算公式
```javascript
if (选择了N个理论) {
  技术指标权重 = 40%
  每个理论权重 = 60% / N
} else {
  技术指标权重 = 100%
}

综合评分 = 技术指标评分 × 技术指标权重
         + 理论1评分 × 理论1权重
         + 理论2评分 × 理论2权重
         + ...
```

## ⚠️ 注意事项

### 使用建议
1. **不要盲目追求高分**
   - 高分不一定准确
   - 关注多个指标的一致性

2. **不同周期可能不同**

   - 以大周期为主

3. **理论不是越多越好**

4. **VOL是重要参考**
   - 任何信号都需要量能确认
