# 🚀 前端版本快速开始指南

## 📦 5秒开始使用

### 最简单的方式
```
1. 找到 jiaoyi.html 文件
2. 双击打开
3. 开始使用！
```

就是这么简单！无需安装任何东西。

## 🎯 第一次使用

### Step 1: 打开应用
双击 `jiaoyi.html` 文件，或者在浏览器中打开它。

### Step 2: 等待连接
- 顶部状态指示器会显示"连接中..."
- 几秒后变为"实时推送中"（绿色圆点）
- 左侧币种列表开始显示数据

### Step 3: 选择币种
- 点击左侧的任意币种（BTC、ETH、SOL等）
- 右侧立即显示该币种的详细信息
- 自动加载K线图

### Step 4: 查看技术分析
- 点击右上角"技术分析"按钮
- 等待几秒计算
- 查看完整的技术指标和交易建议

## 🔧 核心功能使用

### 1️⃣ 实时监控
**默认启动即开启**
- 自动连接币安WebSocket
- 实时推送价格更新
- 左侧列表动态刷新

**操作**：无需操作，自动进行

### 2️⃣ 切换币种
**查看不同币种数据**

**操作**：
```
1. 在左侧币种列表点击任意币种
2. 当前币种会高亮显示（蓝色边框）
3. 右侧立即显示该币种数据
4. K线图自动加载
```

### 3️⃣ 暂停/恢复
**节省资源或暂停推送**

**操作**：
```
1. 点击右上角"暂停"按钮
2. WebSocket断开，停止推送
3. 状态变为"已暂停"（黄色）
4. 点击"恢复"按钮重新连接
```

**用途**：
- 节省网络流量
- 降低CPU占用
- 保持当前数据查看

### 4️⃣ 手动刷新
**暂停模式下更新数据**

**操作**：
```
1. 暂停模式下点击"刷新"按钮
2. 重新获取所有币种数据
3. 更新K线图表
4. 数据立即刷新
```

### 5️⃣ 技术分析
**获取完整技术分析报告**

**操作**：
```
1. 选择要分析的币种
2. 点击"技术分析"按钮
3. 等待3-5秒计算
4. 查看技术指标和交易建议
```

**包含内容**：
- ✅ 5个技术指标（MA、MACD、RSI、BOLL、KDJ）
- ✅ 综合评分（0-100分）
- ✅ 交易方向（做多/做空/观望）
- ✅ 建议止损位
- ✅ 建议止盈位
- ✅ 风险提示

### 6️⃣ 时间周期切换
**查看不同周期K线**

**操作**：
```
1. 在K线图上方选择周期
2. [1h] 1小时K线
3. [4h] 4小时K线
4. [1d] 1天K线
5. 自动重新加载数据
```

## 🔍 数据解读

### 价格卡片
```
当前价格: $110,248.38
  → 最新成交价

24小时涨跌: ▲ +2.39%
  → 相对24小时前的涨跌幅
  → ▲ 上涨（绿色）
  → ▼ 下跌（红色）

24小时最高: $111,190.00
  → 24小时内最高价格
  → 可能是阻力位

24小时最低: $106,304.34
  → 24小时内最低价格
  → 可能是支撑位
```

### 技术指标
```
MA均线
  → 移动平均线
  → 价格在均线上方：看涨
  → 价格在均线下方：看跌

MACD
  → DIF > 0：买入信号
  → DIF < 0：卖出信号

RSI
  → > 70：超买（可能回调）
  → < 30：超卖（可能反弹）
  → 30-70：正常区间

BOLL布林带
  → 价格接近上轨：超买
  → 价格接近下轨：超卖
  → 在中轨附近：正常

KDJ
  → K > 80：超买区域
  → K < 20：超卖区域
  → K上穿D：金叉（买入）
  → K下穿D：死叉（卖出）
```

### 综合评分
```
评分范围: 0-100分

70-100分: 强烈看涨
  → 建议：做多（买入）
  → 可信度：高

55-70分: 偏多
  → 建议：做多（买入）
  → 可信度：中

45-55分: 震荡
  → 建议：观望
  → 可信度：低

30-45分: 偏空
  → 建议：做空（卖出）
  → 可信度：中

0-30分: 强烈看跌
  → 建议：做空（卖出）
  → 可信度：高
```

## 🆘 遇到问题？

### 问题1：页面空白
**解决**：
1. 检查浏览器是否支持（Chrome/Edge最佳）
2. 按F12打开控制台查看错误

**解决**：
1. 检查网络连接
2. 确认币安API可访问
3. 尝试使用VPN（国内用户）
4. 等待自动重连（5秒）

### 问题3：K线图不显示
**解决**：
1. 确保选择了币种
2. 等待数据加载（3-5秒）

4. 检查控制台错误

### 问题4：技术分析没反应
**解决**：
1. 确保K线数据已加载
2. 等待计算完成（3-5秒）
